# Ubuntu Home Lab Project

## Overview
Built and maintained a self-hosted Ubuntu 24.04 home lab on an HP EliteDesk 800 G2 DM.

## Hardware
- HP EliteDesk 800 G2 DM
- 465GB SSD
- 4GB DDR4 RAM (upgrade planned)

## Operating System
- Ubuntu 24.04 LTS

## Services Deployed
- Docker
- Portainer
- Pi-hole
- Grafana
- Prometheus
- cAdvisor
- Nextcloud
- Nginx Proxy Manager
- Homepage Dashboard
- n8n Automation
- Tailscale

## Skills Demonstrated
- Linux Administration
- Docker Container Management
- Network Monitoring
- DNS Filtering
- Reverse Proxy Configuration
- Remote Access Administration
- Infrastructure Documentation

## Results
Successfully deployed and maintained multiple Docker containers providing monitoring, automation, file storage, DNS filtering, and remote administration services.
